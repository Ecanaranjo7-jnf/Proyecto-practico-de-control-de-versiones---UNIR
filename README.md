# Proyecto-practico-de-control-de-versiones---UNIR
Este proyecto es parte de la actividad grupal del Máster Universitario en Bioinformática (UNIR). El objetivo principal es la práctica del uso de Git y GitHub como herramientas de control de versiones y colaboración en proyectos de bioinformática.
“Proyecto práctico de control de versiones para la asignatura Introducción a la Programación Científica (UNIR). Uso de Git y GitHub en un proyecto de bioinformática.”
